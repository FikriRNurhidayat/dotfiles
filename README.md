# Dotfiles

This is where I store my \*nix configuration, such as Neovim, Vim, Tint2, Kitty, XMonad, and other stuff.

## Neovim & Vim

In this configuration file, neovim and vim has completely different setup.
Just in case you might want to use vim, just copy the vimrc,
since the `.config/nvim/init.vim` heavily configured using lua.

If you are a neovim user just copy what's inside the `.config/nvim/init.vim`.
Copying `.vimrc` is unnecessary.
